How It Works

This system uses Machine Learning to estimate indoor CO₂ levels intelligently.

📥 Data Input

Sensor values like Temperature, Humidity, Light and Occupancy are collected in real-time.

🧠 Model Processing

A trained ML model analyzes patterns between environmental factors and CO₂ levels.

📊 Prediction

The model predicts CO₂ concentration (ppm) using learned relationships.

🌿 Air Quality

Output is classified into Good, Moderate, or Poor for easy understanding.
